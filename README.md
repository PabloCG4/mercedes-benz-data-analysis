
## Descripción del Proyecto
Este proyecto se enfoca en la aplicación de técnicas de **Data Science** en la industria automotriz para **Mercedes-Benz**. El objetivo central es explorar patrones en el proceso de compra de vehículos para optimizar la toma de decisiones estratégicas.

## Objetivos Principales
* **Campañas de Marketing Personalizado**: Conectar a los clientes con su vehículo ideal en el momento perfecto.
* **Modelado Predictivo**: Implementación de modelos de **regresión** para predecir la próxima fecha de compra (`Next_repurchase_date`) y modelos de **clasificación** para determinar el tipo de carrocería preferido (`Next_class_bodytype`).
* **Análisis de Wishlist**: Investigar si existe una relación entre añadir un vehículo a la lista de deseos y la conversión final de compra.
* **Impacto del Servicio Post-venta**: Analizar cómo influyen los motivos y la frecuencia de las visitas al taller en la decisión de compra de un nuevo vehículo.

## Metodología
1. **Análisis Exploratorio de Datos (EDA)**: Identificación de relaciones entre tablas y verificación de la calidad de los datos.
2. **Limpieza del Dataset**: Gestión de valores nulos (imputación simple y compleja) y detección/tratamiento de **outliers**.
3. **Unificación de Fuentes**: Integración de datos provenientes de sistemas heterogéneos (MRA y GuK) para eliminar duplicidades.
4. **Feature Engineering**: Creación de nuevas variables agregadas por cliente y vehículo para enriquecer el análisis.
5. **Detección de Patrones**: Identificación de correlaciones entre las variables de comportamiento y los objetivos de negocio.

## Tecnologías
* **R**: Lenguaje principal utilizado para el análisis estadístico y procesamiento de datos.
